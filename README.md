# CodeGuess
CodeGuess is a browser-based puzzle game where players must deduce a hidden numerical code using logic and elimination. Each round generates a random code made up of unique digits, and players receive color-coded feedback after every guess to guide them toward the solution.

No sign-up. No ads. Completely free to play.

# Features
4 Game Modes - Choose between 3, 4, 5, or 6 digit codes
3 Difficulty Levels - Easy, Medium, and Hard, each with different attempt limits
Color-coded Feedback - Instantly know which digits are correct, misplaced, or wrong
Multilingual Support - Available in English, Romanian, and Russian
Responsive Design - Works on desktop, tablet, and mobile

# How to Play

1. Choose a mode — Select how many digits the secret code will have (3–6)
2. Choose a difficulty — This determines how many attempts you get
3. Enter your guess — Type or use the on-screen keyboard
4. Read the feedback:
   - Green - Correct digit, correct position
   - Orange - Correct digit, wrong position
   - Gray - Digit not in the code
5. Keep guessing until you crack the code - or run out of attempts

   - On 3 digits, Easy gives 6 attempts, Medium 4, and Hard only 3.
   - On 4 digits, Easy gives 6, Medium 5, and Hard 4.
   - On 5 digits, Easy gives 8 attempts, Medium 6, and Hard 4.
   - On 6 digits, Easy gives 10, Medium 8, and Hard 6.
